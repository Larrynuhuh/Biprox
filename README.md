Biprox is an implementation of the Generalized Binomial Theorem in JAX. Honestly nothing more than that, that's kind of it. 
For a fresh reminder, the binomial theorem is a centuries old formula which helps us to easily express the expansion of a 
binomial term using a finite series expansion. 

The generalised Binomial Theorem given to us by Sir Isaac Newton provides a generalised formula which allows the exponent to be both negative and in fraction form, unlike the ungeneralised form which can only accept the power of the term as a positive 
integer. It can solve (1 + x)^n, with the drawback that |x| < 1.
